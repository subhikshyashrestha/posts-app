# Posts App – Frontend Assignment

## Project Overview
This is a simple React application that fetches and displays posts from the JSONPlaceholder public API.  
The app focuses on clean code structure, proper data fetching, handling loading/error states, and basic UI/UX best practices.

## Features
- Fetches posts from a public API
- Displays posts in a clean card-based layout
- Handles loading and error states
- “Load More” button to show posts incrementally
- Basic search/filter by post title

## Tech Stack
- React (Vite)
- JavaScript
- CSS
